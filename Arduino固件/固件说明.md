# Arduino 硬件模式 — 烧录说明

## 所需硬件
- Arduino Leonardo 或 Arduino Micro（32U4 芯片）
- Micro USB 数据线

## 安装库
将 `libraries/` 里的 **HID-Project** 和 **MouseTo** 两个文件夹复制到 `文档\Arduino\libraries\`。

## 烧录步骤
1. USB 连接 Arduino → 打开 `roco_firmware_v2_hid.ino`
2. 工具 → 开发板选 Arduino Leonardo（或 Micro）→ 选 COM 端口
3. 点击上传
4. 成功后电脑识别为 USB HID 键盘+鼠标

## 使用
1. 程序切换到 Arduino (硬件) 模式
2. 串口页面选 COM 口 → 连接
3. 状态栏显示 ● Arduino 已连接

## 注意
- 不是 Uno！只有 32U4 芯片支持 USB HID
- 烧录失败按 RESET 按钮
- 波特率固定 115200
