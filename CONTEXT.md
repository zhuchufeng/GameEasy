# CONTEXT.md — 洛克王国减负小助手

本文件定义项目领域术语。所有讨论、issue、代码注释应使用本文档中的术语，保持一致性。

## 游戏概念

### 洛克王国 (Roco Kingdom)
腾讯旗下的网页/客户端宠物养成游戏。本工具针对其 Windows 版本做操作自动化。

---

## 输入系统

### 输入后端 (Input Backend)
向游戏发送键盘/鼠标输入的底层机制。两种可用后端：

- **内核驱动 (Kernel Driver)** — 通过 Interception 驱动注入 PS/2 扫描码，绕过游戏的 RawInput 检测。纯软件方案，需要管理员权限。对应 `UiohookBackend`。
- **Arduino 硬件 (Arduino Hardware)** — 通过串口控制 Arduino Leonardo/Micro 作为物理 USB HID 设备发送按键。游戏完全无法区分，因为就是一个真实的 USB 键盘/鼠标。对应 `ArduinoBackend`。

已废弃：**PC 直连 (PCDirectBackend)** — 使用 `win32api.keybd_event`/`mouse_event`，会被游戏 RawInput 拦截。内核驱动是其完全上位替代。

---

## 自动化功能

### 自动化功能 (Automation)
工具提供的 5 种游戏自动化操作：

| 功能 | 说明 |
|------|------|
| **自动战斗** | 反复按技能键打怪刷级 |
| **跳过剧情** | 自动识别并跳过对话和过场动画 |
| **采矿切宠** | 挖矿时在宠物间循环切换 |
| **一键放生** | 批量释放宠物腾背包空间 |
| **自动丢球** | 自动丢咕噜球捕捉宠物 |

---

## 窗口与坐标

### 游戏窗口 (Game Window)
操作系统层面的完整窗口，包含标题栏和边框。

### 游戏画布 (Game Canvas)
窗口内部的客户区，即游戏实际渲染的区域。

### 画布坐标 (Canvas Coordinates)
以游戏画布左上角为原点 (0,0) 的坐标系统。所有自动化操作（点击、截图匹配、模板检测）均基于画布坐标。

---

## 热键系统

### 热键 (Hotkey)
启动/停止自动化功能的全局按键。由 `pynput` 全局键盘监听器分发，优先级分派：
- 优先级 0：热键捕获控件（设置热键时）
- 优先级 10：自动化功能切换
- 优先级 20：放生坐标捕获（F8-F11）

### 界面快捷键 (UI Shortcut)
在程序界面内切换功能页面的键盘快捷方式（Ctrl+1~7、ESC 停止）。

---

## 宝箱监控

### 宝箱监控 (Box Monitor)
独立于自动化功能的监测引擎。循环扫描游戏画布，检测到宝箱图标时自动截图保存。

状态机流转：
```
ARMED（就绪：扫描中，未发现目标）
  → WAITING（等待：疑似发现宝箱，进一步确认）
    → CAPTURING（截图：确认宝箱，延时后保存截图）
      → COOLDOWN（冷却：截图后休息，避免重复触发）
        → ARMED（回到就绪）
```

使用 OpenCV 模板匹配 (`TM_CCOEFF_NORMED`) + mss 截图。

---

## 图标检测

### 图标检测 (Icon Detection)
"跳过剧情"功能使用 YOLO ONNX 模型实时检测游戏画布上的 UI 元素，根据检测结果执行对应操作：

| 检测对象 | 触发操作 |
|----------|----------|
| **过场图标** | 按 ESC 跳过过场动画 |
| **对话图标** | 按 1 选择对话选项 |
| **确认按钮** | 鼠标点击确认 |

模型输出检测框坐标（画布坐标）+ 类别标签。

---

## 一键放生

### 放生 (Release)
从背包中释放宠物。游戏背包以 **6 列 × 5 行 = 30 格/页** 排列。放生流程：

1. 点击宠物格子
2. 点击确认按钮（4 次确认弹窗）
3. 可选翻页继续

放生坐标通过热键 F8-F11 捕获（画布坐标）。

---

## 自动丢球

### 丢球 (Throw Ball)
捕捉宠物的操作流程：

1. **瞄准间隔**：用户手动瞄准目标宠物（工具等待中）
2. **长按蓄力**：工具按住鼠标
3. **松手丢球**：工具松开鼠标，精灵球丢出
4. 回到步骤 1（循环）

---

## 采矿切宠

### 切宠 (Pet Switch)
采矿时，一只宠物一次只能采一个矿石。需要按顺序轮流派出每只宠物：

- 默认切换顺序：2 → 3 → 4 → 5 → 6（宠物技能栏位）
- 监听鼠标松开事件后触发切换
- 循环重复

---

## 反检测

### 随机延迟 (Random Delay)
在每次自动化操作间隔上附加随机毫秒数（可配置最小/最大值），使操作节奏不规则，降低被游戏反作弊检测为脚本的概率。
