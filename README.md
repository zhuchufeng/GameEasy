# 洛克王国减负小助手 v1.2

洛克王国 PC 端全功能自动化工具。支持内核驱动和 Arduino 硬件两种输入方案。

## 功能列表

| 功能 | 说明 | 热键(可自定义) |
|------|------|:--:|
| 自动战斗 | 反复按技能键挂机刷怪 | F1 |
| 跳过剧情 | YOLO 深度学习识别对话/过场，自动跳过 | F2 |
| 采矿切宠 | 监听鼠标松手自动循环切换精灵 | F3 |
| 一键放生 | 6×5 网格批量释放宠物 | F4 |
| 自动丢球 | 按住/松开鼠标循环丢球捕捉 | F5 |
| 宝箱检测 | YOLO 全屏监测宝箱，自动截图保存 | 可配 |
| 互访炫彩 | 双窗口自动化互访 + 小地图检测 PT/HB | 按钮启动 |

## 使用说明

### 1. 安装 Interception 驱动（内核驱动模式）
1. 以管理员身份运行 `install-interception.exe`
2. 安装完成后**重启电脑**
3. 确保 `interception.dll` 与程序在同一目录

### 2. 启动程序
- **开发版**: 以管理员身份运行 `启动(管理员).bat`
- **用户版**: 双击 `洛克王国减负小助手.exe`（自动请求管理员权限）

### 3. 基本操作
1. 点击顶部下拉框选择游戏窗口
2. 默认「内核驱动」模式，状态栏显示 ● 驱动就绪
3. 切换到对应功能页面，配置参数
4. 点击开始或按热键启动

### 4. 全局热键
- 鼠标下侧键 = 全局紧急停止
- 全局停止热键可在顶部栏自定义（默认 F8）
- 各功能启停热键可在对应页面自定义
- 所有热键在游戏中也有效（需管理员权限）

### 5. 放生坐标设置
1. 进入放生页面，先选择游戏窗口
2. 鼠标移到背包第一个宠物上 → 按 F9
3. 鼠标移到背包最后一个宠物上 → 按 F10
4. 鼠标移到确认按钮上 → 按 F11
5. 鼠标移到翻页按钮上 → 按 F12
6. 点击"开始一键放生"

### 6. 互访炫彩
1. 选择窗口1（操作窗口）和窗口2（目标窗口）
2. 阶段模型自动从 models/visitor/ 加载
3. 小地图模型自动从 models/colorful.onnx 加载
4. 框选窗口2的小地图区域
5. 配置 Server酱 SendKey 接收微信通知（可选）
6. 点击开始

### 7. 跳过剧情
1. 模型自动从 models/skip/skip_model.onnx 加载
2. 调整置信度阈值和检测间隔
3. 点击开始或按热键启动
4. 浮窗显示实时检测状态

## 目录结构

```
├── 洛克王国减负小助手.exe    # 用户版主程序
├── interception.dll          # Interception 驱动 DLL
├── models/                   # YOLO 模型文件
│   ├── skip/skip_model.onnx  # 跳过剧情模型
│   ├── visitor/*.onnx        # 互访炫彩阶段模型
│   ├── colorful.onnx         # 小地图检测模型
│   ├── boxcheck.onnx         # 宝箱检测模型
│   └── boxcheck_seg.onnx     # 宝箱检测模型(分割)
├── roco_auto/data/           # 配置文件目录
└── 使用说明.txt              # 使用说明
```

## 管理员开发版

开发版为完整源代码，需安装 Python 3.10+ 及依赖：

```bash
pip install -r requirements.txt
python -m roco_auto
```

## 注意事项

- 内核驱动模式**必须**以管理员身份运行
- Interception 驱动安装后需要**重启电脑**
- 如键盘鼠标注入失效，尝试重启电脑
- 放生坐标使用绝对屏幕坐标，更换分辨率后需重新捕获
- 互访炫彩需要两个游戏窗口同时运行
