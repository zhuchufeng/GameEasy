# 洛克王国减负小助手 v1.2 — 部署说明

## 用户版 (EXE 一键启动)

1. 复制整个 `用户使用版` 文件夹到目标电脑
2. 以管理员身份运行 `驱动/install-interception.exe` 安装驱动
3. **重启电脑**
4. 双击 `洛克王国减负小助手.exe`（自动请求管理员权限，UAC点"是"）
5. 顶部选择游戏窗口 → 配置热键 → 开始使用

**注意**: `interception.dll` 和 `models/` 必须与 EXE 同目录。

## 开发版 (Python 源码)

1. 安装 Python 3.10+
2. `pip install -r requirements.txt`
3. 双击 `启动(管理员).bat` 或命令行 `python -m roco_auto`

## Arduino 硬件模式（可选）

1. Arduino IDE 打开 `Arduino固件/roco_firmware_v1.ino`
2. 安装库：HID-Project + MouseTo（库管理器搜索安装）
3. 烧录到 Arduino Leonardo/Micro
4. 程序中选择 Arduino 模式 → 串口页面连接

## 模型文件

| 路径 | 用途 |
|------|------|
| models/skip/skip_model.onnx | 跳过剧情 (yolo11, 3类) |
| models/visitor/*.onnx | 互访炫彩 5阶段模型 |
| models/colorful.onnx | 小地图炫彩检测 (PT/HB) |
| models/boxcheck.onnx | 宝箱检测 |
| models/boxcheck_seg.onnx | 宝箱检测(分割) |

## 常见问题

- **键盘鼠标不生效**: 重启电脑
- **驱动未就绪**: 未以管理员运行，或驱动未安装
- **检测不到**: 检查 models/ 目录完整性，调整置信度阈值
